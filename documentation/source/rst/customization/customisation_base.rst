Customization Overview
======================

To customize the web application, you need to provide your own resources in the following structure:

.. code-block:: text

		config.js
		data/
			├── line.json
			├── station.json
		image/
			service/
				├── local.svg
				├── express.svg
			├── favicon.ico
			├── logo.svg
			├── map.svg
			├── train-animation.svg
		style/
			├── map-font.css
			├── text-font.css
